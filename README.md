# SmartLoan Platform

A full-stack microloan management system.

## Setup

1. **Backend**:
   ```bash
   cd server
   npm install
   npm run dev
   ```
   Runs on http://localhost:5001

2. **Frontend**:
   ```bash
   cd client
   npm install
   npm run dev
   ```
   Runs on http://localhost:5173

## Features
- User/Admin Dashboards
- Real-time loan status updates
- Loan application and offers
